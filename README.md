# EvernightBoard

寄长夜之灵犀，化万语为此间。

Entrusting the soul's connection to the deep night, transforming ten thousand voices into this space.

常夜（とこよ）に霊犀（れいさい）を託（たく）し、万言（ばんげん）をこの方寸（ほうすん）に。

EvernightBoard 是一款通过预设图文，在触屏受限时为您延续沟通的展示辅助工具。

EvernightBoard is an assistive tool for continuous communication via preset text and images when touch input is limited.

EvernightBoardは、タッチパネルの不調時でも、登録済みの画像やテキストで意思疎通を支える支援ツールです。
